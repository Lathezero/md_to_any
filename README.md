# MD To Any

一个强大的工具，用于将Markdown文件转换为各种其他格式，如Word文档、HTML等。

## 功能

- Markdown转Word文档
- Markdown转HTML
- 支持图片嵌入和处理
- 简单易用的Web界面

## 安装

```bash
npm install
```

## 使用方法

```bash
npm start
```

然后在浏览器中打开 http://localhost:3000

## 配置

修改`package.json`文件中的配置项以适应您的需求。
